# 末法时代的内景地与旧术荣光

> 一拳碎虚空，于高科技战场重塑武道辉煌

## 小说简介

公元 2147 年，科技霸权确立，星际舰队巡逻在太空轨道，能量武器普及到每个角落。古武"旧术"被视为淘汰品，仅存于博物馆和极少数传承者口中。

大学生王煊在一次意外中，精神进入了远古大能遗留下的"内景地"。在这里，他能亲历强者生平，感悟失传神通。搬山劲、踏云步、九转玄功……一门门惊天地泣鬼神的神通，被他从内景地带回现实。

当星际舰队压境，新型武器失效时，他凭借从内景地带回的失传旧术，一拳碎虚空，于高科技战场重塑武道辉煌。

## 核心看点

- **拳头对科技的暴力美学**：肉身破音障，拳风伤人于十步之外
- **新旧文明的碰撞**：能量武器 vs 肉身神通，星际舰队 vs 一拳破敌
- **力量体系的反差感**：被世人抛弃的"落后文化"，竟是拯救人类的关键

## 世界观设定

| 元素 | 描述 |
|------|------|
| **时代背景** | 公元 2147 年，科技霸权时代 |
| **内景地** | 远古大能留下的精神空间，共九重，对应九个修行境界 |
| **旧术体系** | 以肉身证道的古武修行，核心是搬山劲、踏云步等神通 |
| **科技势力** | 科技议会、星际舰队、能量武器 |

## 主要角色

- **王煊**：21 岁大学生，意外获得内景地传承，旧术修行者
- **林小雨**：科研系天才少女，旧术传承者后代，王煊的重要助力
- **玄穹真君**：远古大能，内景地主人，王煊的引导者
